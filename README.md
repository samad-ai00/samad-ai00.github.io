# Abdul Samad Satti — Portfolio

A high-performance, aesthetically driven portfolio website built with Astro 6 and Tailwind CSS. Designed for AI/ML Engineers and Full-Stack Developers, featuring a sleek dark mode, smooth transitions, and an advanced project showcase.

---

## Features

- **Built with Astro 6**: Ultra-fast performance with zero-JS by default.
- **AI/ML & Data Showcase**: Dedicated sections highlighting Deep Learning, Computer Vision, and Data Analysis insights.
- **Modern Aesthetics**: Premium design with dark/light mode support.
- **Fully Responsive**: Optimized for all devices from mobile to desktop.
- **Project Grid & Case Studies**: Deep dives into complex technical projects (like 3D Medical AI processing).
- **Blog Engine**: Built-in support for technical articles and AI research breakdowns.
- **Contact System**: Integrated contact form for professional inquiries.
- **Optimized Assets**: Image optimization and lazy loading.

---

## Tech Stack

- **Frontend Framework:** [Astro](https://astro.build/) & [Tailwind CSS](https://tailwindcss.com/)
- **Backend & Integration:** [Laravel / PHP](https://laravel.com/)
- **AI/ML Focus:** Python, Deep Learning, Computer Vision, NLP
- **Icons:** [Lucide React](https://lucide.dev/) (via Astro)
- **Deployment:** [Vercel](https://vercel.com/)
- **Virtualization:** [Podman](https://podman.io/) & [Podman Compose](https://github.com/containers/podman-compose)
- **OS Compatibility:** Optimized for [Fedora](https://fedoraproject.org/) and other RHEL-based distributions
- **Animations:** CSS Transitions & Intersection Observer
